# Add wifi to your pelpro 130
## Parts: 
- EMAX Servo ES08A (1)
- ELEGOO ESP-32 (Or similar) (1)
- 8mm x 2mm Round Magnets (3)

## 3D print the parts
- Servo Motor / ESP case
- Servo Spur gear
- PelPro Gear

## Flash ESP32
- Use your IP Address / Subnet / DNS
- Create your password or omit it if needed
